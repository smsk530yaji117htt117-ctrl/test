# 決定パッケージ テンプレート（Go/No-Go を「選ぶだけ」にする）

人間判断が要る項目を、矢嶋さんが**材料集めなしで選ぶだけ**にするための1枚テンプレ。
改善エンジンのしくみ3（判断ゲートの摩擦削減）。code-only で潰せない項目（戦略・インフラ・
スキーマ・Render・新規 Cloud Routine）は、実装の前にこの形へ"前裁き"して届ける。

> 使い方: このテンプレをコピーして `docs/decisions/<番号>-<slug>.md` を作る。
> 窓口OS は埋めるだけ。矢嶋さんは「推奨どおり」か「別の選択肢」かを一言で返す。
