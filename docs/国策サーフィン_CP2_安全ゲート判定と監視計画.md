## 3. ハードルレートの計算（なぜ返済が先か）

借入があれば、同額返済で**確実に**消える利息を投資の最低ハードルレートとして扱う、というのがあなたのルール。

| 前提金利 | 200万円に対する年間利息 | 月3万円返済の「確実な」利回り |
|---|---|---|
| 年15%（カードローン／リボ標準） | **約30万円/年** | **年15%相当（確実・非課税）** |
| 年18%（キャッシング上限級） | 約36万円/年 | 年18%相当 |
| 年12%（分割払い） | 約24万円/年 | 年12%相当 |

**月3万円の使い道の比較：**
- 返済に充てる → **年12〜18%の確実なリターン**（利息の消滅・複利の停止）。リスクゼロ、税金なし。
- 1年保有の骨太テーマ株に充てる → **年12〜18%超を、確実に、税引き後で**上回らないと負け。短期では市場の上下に支配され、確実性はない。

さらに月3万は利息（年15%なら月約2.5万）をようやく上回る程度。**投資に回すと元本200万がほぼ減らず、利息が積み上がり続ける**。これは典型的な借金スパイラルの入口。

> 結論の数値はレンジで提示している。精密に見せかけない。ただし**どの金利前提でも、1年・課税口座の株式投資が確実な15%前後の返済利回りを安定的に上回る根拠はない**。

## 4. 「1年で骨太の旨みだけ」が、今は特に弱い理由

あなたの狙い（骨太2026の上昇だけ短期で取る）は、CP1で判明した政策フェーズと真っ向からぶつかる。

1. **材料出尽くしリスクが最大化する局面。** 骨太2026は**フェーズB＝原案、閣議決定は6月下旬〜7月（数週間以内）**。テーマ株は「発表前に上がり、発表で出尽くす」のが定型。今から1年保有で買うと、**最も高い所で掴み、決定後の剥落を食らう**確率が高い。
2. **織り込み済みの可能性。** 防衛・AI半導体など有名テーマは既に株価が先行。あなたのルールでも「発表前に上昇しているテーマは減点」。
3. **政策→予算→受注→利益→株価には時間差がある。** 骨太（方針）の段階では多くが「検討」止まり。実際の予算成立・採択・受注は**今夏〜来年度**。1年では政策の利益寄与が株価に乗り切らないテーマが多い。
4. **生活防衛資金ゼロ×収支トントン**だと、株価が一時的に下げただけで耐えられず、最悪の所で売る。短期イベント投資は値動きが荒く、この体力では破綻しやすい。

→ 「良いテーマ」であることと「今のあなたが今買うべき」かは別。**今の最良の"トレード"は、確実に年15%前後で回るカード返済**。

## 6. 投資を「再起動」する条件（この順番で）

1. **生活防衛資金を確保**（最低でも生活費3〜6か月分を現金で）。
2. **カード借入200万を完済**（または高金利分を最優先で消す）。
3. **収支を黒字化**し、株に回す資金が「失っても生活が揺らがない余剰」になる。
4. **許容最大損失額（円・％）を自分で決める**。
5. これらが揃ったら、CP1の監視リストを使って**政策具体化（予算成立・採択・受注）に合わせて段階的に検討**を再開。

→ 上記が揃うまでは、骨太がどんなに盛り上がっても**見送り**。これは機会損失ではなく、確実な15%返済を取りに行く「勝ちトレード」。
