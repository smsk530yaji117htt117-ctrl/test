# 補足調査：過去の「国策テーマ先頭株」の1年後はどうなったか

> 目的：「骨太の方針／国策テーマで最初に上がり始めた銘柄を買い、1年持つ」戦略が実際に機能したかを過去事例で検証する。
> 投資助言ではなく事実の整理。基準日 2026-06-18 JST。価格は各報道・株価サイト公表値（公開日基準）。**確認できない精密値は推測で埋めていない。**

---

## 0. 一行まとめ

**「最初に上がり始めた銘柄を1年保有」は、過去事例では期待値が低い。** 勝った例（三菱重工）は記憶に残るが、その裏に同規模の惨敗例（レノバ、レーザーテック高値掴み、利益なきゼネコン）が隠れており、**生存者バイアス**が極めて強い。先頭株ほど株価が利益に先行しており、**材料出尽くし・期待剥落の直撃を受けやすい。**

---

## 1. 骨太の方針の歴代テーマ（概観）

骨太の方針は2001年開始。毎年6月頃に閣議決定し、年末の予算編成を方向づける。時の政権の目玉が入る。

| 年 | 政権 | 代表的テーマ |
|---|---|---|
| 2013 | 第2次安倍 | アベノミクス三本の矢、機動的財政（公共投資10兆円）、**国土強靱化** |
| 2020 | 安倍 | コロナ対応、デジタル化、グリーン |
| 2021 | 菅 | グリーン（GX/脱炭素）、デジタル、**半導体**、**洋上風力** |
| 2022 | 岸田 | 新しい資本主義、人への投資・賃上げ、**防衛費増額** |
| 2023 | 岸田 | 経済・社会・ジェンダー、GX、半導体、異次元の少子化対策 |
| 2025 | 石破 | 賃上げ（実質1%目標） |
| 2026 | 高市 | 責任ある積極財政、日本成長戦略17分野（策定中） |

出典：Wikipedia「骨太の方針」、経済同友会意見書、各社まとめ。

## 2. ケーススタディ（先頭株の1年後）

### ✅ 成功例：三菱重工業【7011】＝防衛（骨太2022／防衛費43兆円 2022年12月）

| 項目 | 内容 |
|---|---|
| 政策 | 2023〜27年度の防衛費を5年総額**43兆円**（前中期27兆円の約1.6倍）へ |
| 株価 | 2020〜21年の安値圏から2026年に一時5,000円超、**数年で約10倍** |
| 利益への伝達 | 防衛・宇宙の**受注高 5,571億円→1兆8,781億円**（実数で急拡大） |
| 注意点 | 途中で「**業績は過去最高なのに約2割下落**」する局面が複数回。2024/4に1:10株式分割 |

**教訓：** これは「政策→受注→利益→株価」が本当に繋がった本物の成功。ただし**①政策が実需(受注)に転換し、②買った時点でまだ織り込み切っていなかった、③そして1年ではなく複数年、しかも途中の▲20%調整に耐えた**——この3条件が揃って初めて報われた。「1年で旨み取り」では途中の急落で振り落とされた可能性が高い。
出典：かぶリッジ、moomoo、マネクリ（マネックス）、三菱重工IR。

---

### ❌ 失敗例1：レノバ【9519】＝洋上風力（GX、骨太2021の目玉）

国策「カーボンニュートラル／洋上風力」の**純粋関連の先頭株**として2021年に急騰。だが**実際の政府調達（入札）で敗北した瞬間に崩壊**。

| 時点 | 株価／出来事 |
|---|---|
| 2021/9 | 上場来高値 **6,390円** |
| 2021/12/24 | 秋田・千葉沖の洋上風力公募で**三菱商事連合に敗北、選定漏れ** |
| 2021/12/27〜28 | **2営業日連続ストップ安**。3日で時価総額**約1,900億円**消失 |
| 2022/1/20 | 終値**1,587円**＝高値の**4分の1以下**。13営業日連続安値更新 |

**教訓：** テーマの「先頭」＝純粋関連株は、期待が最も乗っている分、**政策の現物（調達結果）が出た時の下振れ直撃が最大**。しかも落札価格が予想を大幅に下回り、業界全体の採算懸念に発展。
出典：日本経済新聞、Bloomberg、東洋経済四季報オンライン。

> さらに皮肉なことに、**勝った側の三菱商事も**安値落札がたたり2024年に洋上風力から撤退・巨額損失。**「受注を取る＝儲かる」ではない**ことの典型。

---

### ❌ 失敗例2：レーザーテック【6920】＝半導体（国策×世界半導体ブーム）

半導体（国策テーマかつ世界的物色）の**象徴的先頭株**。期待先行で買った高値掴み層が壊滅。

| 時点 | 株価／出来事 |
|---|---|
| 2024/5/20 | ピーク **45,500円** |
| 2025/7 | **13,700円**＝ピークから**約7割安**（半値以下） |
| 業績 | FY2025/6は**過去最高益**（営業益+51%）。なのに株価は暴落 |
| 背景 | EUV関連需要のピークアウト懸念、受注鈍化、**受注開示を廃止** |

**教訓：** **業績が過去最高でも、期待（バリュエーション）が先に行き過ぎていれば株は下がる。** あなた自身のルール「株価が業績成長より先行していないか」を地で行く事例。
出典：日本経済新聞、株式新聞Web、東洋経済四季報。

### ⚠️ 「利益なき繁忙」例：ゼネコン＝国土強靱化（アベノミクス2013）

| 項目 | 内容 |
|---|---|
| 政策 | 2013年、公共投資10兆円、国土強靱化・五輪・リニアで建設活況 |
| 売上 | 受注は急増（繁忙） |
| 利益 | 2013〜14年は過当競争で採算度外視受注＝「**利益なき繁忙**」 |
| 株価 | 利益が伴わず、最高益更新後も2018年以降は談合・不正等で下落圧力 |

**教訓：** **売上（受注）増加 ≠ 利益増加。** 政策で仕事は増えても、価格決定力がなければ利益に残らない。あなたのルール「低利益率案件しか増えない」減点項目そのもの。
出典：楽天証券トウシル、日興フロッギー、東洋経済。

## 3. 4事例から見える共通パターン

| パターン | 内容 | 該当例 |
|---|---|---|
| 生存者バイアス | 成功(三菱重工)は語られ、失敗は忘れられる。「国策に売りなし」は選択バイアス | 全体 |
| 先頭株ほど危険 | 最初に最も上がる純粋関連株＝期待が最も乗る＝出尽くし・剥落の直撃が最大 | レノバ、レーザーテック高値 |
| 売上↑ ≠ 利益↑ | 受注が増えても採算が悪ければ報われない | ゼネコン、三菱商事(洋上風力) |
| 調達結果で明暗 | 「テーマ」段階と「実際にお金が出る」段階は別。落札・受注で勝者と敗者が確定 | レノバ vs 三菱商事 |
| 勝者も複数年＋我慢 | 本物の勝ち(三菱重工)も1年でなく数年、途中▲20%耐久が前提 | 三菱重工 |

## 4. あなたの当初プラン（1年で骨太の旨み取り）への含意

過去事例は、**あなたが狙う形（テーマ先頭株を発表前後に買い、1年保有）が、最も負けやすいパターン**であることを示している：

- 1年保有では、勝ち筋(三菱重工型)に必要な「複数年×途中▲20%耐久」が満たせない。
- 先頭株を狙うほど、レノバ／レーザーテック型の**高値掴み→剥落**確率が上がる。
- 骨太2026は**いま原案段階で閣議決定が数週間以内**＝まさに「事実で売られる」直前。
- **生活防衛資金ゼロ＋カード借入200万**の体力では、レノバ型の急落（数日で1/4）やレーザーテック型の▲70%に耐えられず、最悪値で投げる。

→ 補足調査は、**先の安全ゲート結論（借入返済優先・現時点で投資見送り）を裏付ける**。この戦略を将来やるとしても、勝てる型は「**先頭株を1年持つ**」ではなく「**実需(受注・採択)を確認してから、複数年・分散で**」入る形。

## 5. 限界・未確認

- 三菱重工の「発表ちょうど1年後」の精密な分割調整リターンは本調査で確定できず（日経電子版/Yahoo!ファイナンスの過去10年株価で要確認）。方向（大幅上昇）と受注実数は確認済み。
- 事例は代表例の抽出であり、全骨太テーマの網羅統計ではない。厳密な期待値検証には全テーマ×先頭株×1年後リターンのデータセットが必要（次段階の課題）。
