## 2. 作業用プロンプト集

### Step 1: 文字起こしの取得方法

#### 方法A: 無料ツール（推奨）
```
1. https://notegpt.io/youtube-transcript にアクセス
2. YouTube URLを貼り付け
3. 日本語の文字起こしをコピー

※ 他の選択肢:
- https://kome.ai/youtube-transcript（無料）
- Googleの「YouTube字幕」からコピー（手動）
```

#### 方法B: 字幕がない動画の場合
```
1. yt-dlp等で音声をダウンロード
2. Whisper（OpenAI）で文字起こし
   - 無料: https://huggingface.co/spaces/openai/whisper
   - ブラウザ上で完結
```

### Step 2: ブログ記事化プロンプト

```
以下はYouTube動画の文字起こしです。
これを読みやすいブログ記事にリライトしてください。

## 指示
- 文字数: 3,000〜5,000文字
- 文体: ですます調
- 構成: タイトル(h1) → 導入文 → h2見出し3〜6個 → まとめ
- 話し言葉を自然な書き言葉に変換してください
- 「えー」「あのー」等のフィラーは削除
- 重複している内容は統合してください
- 要点は箇条書きや表を使って整理してください
- SEOを意識して、主要キーワードを自然にタイトル・見出し・本文に含めてください
- 動画内の具体例やエピソードはできるだけ残してください

## 文字起こし
---
（ここに文字起こしを貼り付ける）
---
```

## 3. 品質チェックリスト（納品前に確認）

```
□ 誤字脱字がないか
□ 話し言葉が残っていないか（「〜じゃないですか」等）
□ 「えー」「あのー」等のフィラーが残っていないか
□ 見出し構成が論理的か
□ 同じ内容の繰り返しがないか
□ 固有名詞（人名・サービス名等）が正しいか
□ 動画の趣旨と記事の内容がずれていないか
□ 文字数が3,000文字以上あるか
□ 読んで自然な日本語か（声に出して確認）
```
