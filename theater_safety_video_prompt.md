# 映画館アワード 開場前安全動画 制作プロンプト

## 概要

会社表彰式（アワード）の映画館開場前に上映する、災害発生時の注意事項を伝える動画の制作プロンプトです。
「STOP映画泥棒」のような映画館文化に溶け込む演出スタイルで、映画体験を壊さず、かつ確実に情報を届けることを目的とします。

---

## 動画制作プロンプト（AI動画生成・制作ディレクション共通）

---

### ■ トーンとスタイル

**参照イメージ:** 日本の映画館で上映される「STOP映画泥棒」シリーズ
- 映画のワンシーンのような映像クオリティ
- BGMは映画音楽風（オーケストラ調、または緊張感と温かみのバランスをとったシネマティックBGM）
- 過度に怖くならないよう、落ち着いたトーンで「頼れる情報」として届ける
- 映画館の大スクリーンに映えるシネスコープ（横長）のアスペクト比（2.39:1 または 16:9）
- 全体の尺：**90秒〜120秒**

---

### ■ シーン構成・演出プロンプト

---

#### 【オープニング：0〜10秒】

**映像:**
映画館の客席から見た視点。豪華なスクリーンが暗転から徐々に明るくなる。フィルムグレインのテクスチャをオーバーレイして、映画フィルムが回るような演出。

**テキスト（スクリーン中央に映画タイトルカード風に表示）:**
```
本日のアワードを安全にお楽しみいただくために
```

**BGM:** 静かなオーケストラの導入。映画の予告編風のドラムロール。

---

#### 【シーン1：非常口確認 ／ 10〜40秒】

**映像コンセプト:**
スタイリッシュなシルエットの人物（映画のスパイ映画的なビジュアル）が、映画館の客席を歩き、非常口の方向を指し示す。非常口のサインが発光して目立つ演出。視聴者目線の主観ショット（POV）で「非常口を目で確認する」動作を表現。

**ナレーション（落ち着いた声、男性または女性）:**
> 「本日の避難経路は、事前にメール（またはLINE）にてお送りしています。
> 今一度、近くの非常口の位置を——
> 目視で——確認してください。」

**テロップ（映画字幕スタイルで画面下部に表示）:**
```
避難経路は事前のメール（LINE）をご確認ください
お近くの非常口を「目視で」ご確認ください
```

**演出メモ:**
「目視で」のタイミングで、映像が一瞬ズームインして非常口サインにフォーカス。カメラ目線で「見てください」というジェスチャーを添えてもよい。

---

#### 【シーン2：慌てず指示に従う ／ 40〜75秒】

**映像コンセプト:**
映画のクライマックスシーンを彷彿とさせる演出。ざわつく映画館内（群衆シルエット）に、スタッフ（制服着用）がさっそうと現れ、落ち着いた所作で手を広げ誘導する。スローモーション演出で「冷静さ」を表現。

**ナレーション:**
> 「万が一、災害が発生した際は——
> 慌てて、むやみに逃げ出さないでください。
> 館内放送、施設スタッフ、そしてアワード委員の指示に従い、
> 落ち着いて行動してください。」

**テロップ（映画エンドクレジット風、縦スクロールまたはフェードイン）:**
```
慌てず、むやみに逃げ出さない

■ 館内放送
■ 施設スタッフ
■ アワード委員

上記の指示に従ってください
```

**演出メモ:**
テロップは3項目それぞれが順番にフェードインする演出で「しっかり読んでもらう」構成に。BGMがやや高揚してから、落ち着きを取り戻すように展開。

---

#### 【シーン3：身の安全を最優先に ／ 75〜105秒】

**映像コンセプト:**
シネマティックな夕焼けや光の映像をバックに、シンプルで力強いメッセージを大きく表示。映画のエンドカード的な演出で、温かみと力強さを表現。

**ナレーション:**
> 「最後に——最も大切なことをお伝えします。
> 緊急時は、備品や荷物は気にしなくて構いません。
> あなた自身の安全を、最優先に行動してください。」

**テロップ（画面中央に大きく、映画タイトル風）:**
```
一番大切なのは
あなた自身の安全
```

**演出メモ:**
このシーンのみ音楽が静まり、ナレーションの声とともに静寂が生まれる演出で「重み」を伝える。

---

#### 【エンディング：105〜120秒】

**映像:**
映画フィルムが巻き取られるような映像でフェードアウト。ロゴやクレジット風に主催者名（アワード名）を表示。

**テキスト:**
```
29期2Q組織別表彰
本日のアワードを安心してお楽しみください
主催：レイスグループ
```

**BGM:** 温かみのあるエンディングテーマで締めくくり。

---

## 会場別カスタマイズ注意事項

動画は会場ごとに一部テロップ・ナレーションを差し替えてください。

---

### 会場A：丸の内ピカデリー（東京）

| 項目 | 内容 |
|------|------|
| 所在地 | 東京都千代田区 |
| 災害時の指示役 | 劇場スタッフ |
| 避難場所 | **現地（館内）に留まる**（高層階のため）|
| 津波発生時 | 同様に現地に留まる |

**シーン2のナレーション差し替え案:**
> 「本施設は高層階に位置しており、津波発生時を含め、原則として現地にとどまることが安全です。劇場スタッフの指示に従ってください。」

---

### 会場B：ユナイテッド・シネマ豊洲（東京）

| 項目 | 内容 |
|------|------|
| 所在地 | 東京都江東区豊洲 |
| 災害時の指示役 | 劇場スタッフ |
| 避難場所 | **現地（館内）に留まる** |
| 津波発生時 | 同様に現地に留まる（建物の高層階） |

**参考資料:** 江東区津波ハザードマップ（南海トラフ対応版）

**シーン2のナレーション差し替え案:**
> 「豊洲エリアでは、津波発生時も含め、本施設内にとどまることが推奨されています。劇場スタッフの指示を必ずご確認ください。」

---

### 会場C：TTホール（大阪）

| 項目 | 内容 |
|------|------|
| 所在地 | 大阪府大阪市（大阪城公園近隣） |
| 災害時の指示役 | 劇場スタッフ |
| 避難場所 | **大阪城公園一帯** |
| 津波発生時 | 同様に大阪城公園一帯へ |

**シーン2のナレーション差し替え案:**
> 「災害・津波発生時は、劇場スタッフの指示に従い、大阪城公園一帯へ避難してください。」

**テロップ差し替え:**
```
避難場所：大阪城公園一帯
（津波発生時も同様）
```

## 制作・演出チェックリスト

- [ ] 映画館の雰囲気に合わせた映像クオリティ（フィルムグレイン・シネマティックカラーグレーディング）
- [ ] BGMは映画音楽風（著作権フリー素材を使用、または原曲制作）
- [ ] テロップは映画字幕・タイトルカードのフォントスタイル（例：セリフ体、白抜き、シャドウあり）
- [ ] ナレーションは落ち着いたプロのトーン（過度な警告口調にしない）
- [ ] 動画尺：90〜120秒以内
- [ ] 会場ごとにナレーション・テロップを差し替えたバージョンを各1本制作
- [ ] 開場前SEまたはBGMとの接続タイミングを現地で確認
