# Claude Code 導入提案書
## 画像文字起こし業務の自動化

**作成日：** 2026年4月14日  
**対象業務：** 名刺・現場看板の画像文字起こし業務

---

## 1. 背景・目的

現在、全国のエージェントから毎月約45,000枚の画像（名刺・現場看板）がGoogle Driveに格納されており、担当者が手動でAPIに投入・スプレッドシートに転記・目視確認を行っている。

この業務に **Claude Code** を導入し、一連の作業を自動化することで、担当者の工数を大幅に削減することを目的とする。

---

## 2. 現状の業務フロー

```
① Google Drive フォルダを開く
      ↓
② 10枚単位で手動選択し、Gemini API に投入
      ↓
③ 文字起こし結果をコピー
      ↓
④ Google スプレッドシートに貼り付け
      ↓
⑤ 全件を目視で誤字・脱字・漏れチェック
```

### 現状の業務規模（月間）

| 業務 | 画像枚数 | バッチ数（10枚単位） | 手作業件数 |
|------|---------|-------------------|-----------|
| 名刺文字起こし | 約 15,000 枚 | 約 1,500 回 | 1,500 回の投入・貼り付け |
| 現場看板文字起こし | 約 30,000 枚 | 約 3,000 回 | 3,000 回の投入・貼り付け |
| **合計** | **約 45,000 枚** | **約 4,500 回** | **4,500 回の手作業 ＋ 全件目視確認** |

### 現状の課題

- **工数が膨大：** 月4,500回の投入・貼り付け作業が発生している
- **目視確認の負担：** 45,000行すべてを人の目でチェックしている
- **ヒューマンエラーのリスク：** 手動貼り付けによるズレ・見落としが発生しうる
- **処理タイミングの制約：** 担当者が対応できる時間帯にしか処理が進まない

---

## 3. 自動化後の業務フロー

```
【自動実行（夜間バッチ等）】

① Google Drive フォルダを自動スキャン
      ↓
② 未処理ファイルを検出・10枚単位で自動分割
      ↓
③ Gemini API へ自動投入・結果取得
      ↓
④ Google スプレッドシートへ自動書き込み
      ↓
⑤ 自動チェックで疑わしい行にフラグを付与
      ↓
⑥ 完了通知を送信（件数・要確認数を報告）

【担当者の作業】

⑦ フラグが付いた行のみ目視確認
```

---

## 4. 自動化の詳細

### 4-1. バッチ処理の自動化

| 項目 | 内容 |
|------|------|
| 対象フォルダ | Google Drive 内の名刺フォルダ／看板フォルダ |
| 処理単位 | 名刺：10枚／回、現場看板：10枚／回 |
| 実行タイミング | 任意（夜間自動実行も可） |
| エラー対応 | 失敗バッチを自動リトライ・ログ記録 |
| 進捗管理 | 処理済み・未処理を自動でステータス管理 |

### 4-2. 自動品質チェック

文字起こし後、以下の項目をスクリプトで自動検証し、問題のある行にフラグを付与。

**名刺（自動チェック項目）**

- 氏名・会社名が空欄
- 電話番号の桁数が不正（10桁未満など）
- メールアドレスの形式が不正（@ なしなど）
- 文字化けの疑いがある記号の混入

**現場看板（自動チェック項目）**

- 施工会社名が空欄
- 住所・現場名が空欄
- 読み取り文字数が異常に少ない（画像不鮮明の可能性）

### 4-3. 処理結果の通知・レポート

処理完了時に自動でレポートを出力。

```
【処理完了レポート（例）】
実行日時  ：2026年4月14日 03:12
名刺      ：1,500 件処理完了 / 要確認 42 件
現場看板  ：3,000 件処理完了 / 要確認 65 件
エラー    ：3 件（詳細はログ参照）
```

## 5. 導入による効果

| 観点 | 現状 | 自動化後 | 削減効果 |
|------|------|---------|---------|
| 投入・貼り付け作業 | 月 4,500 回（手動） | 0 回（自動実行） | **100% 削減** |
| 目視確認件数 | 約 45,000 行 | 約 4,500〜9,000 行（推定） | **80〜90% 削減** |
| 処理タイミング | 勤務時間内のみ | 夜間・休日も自動実行可 | **処理速度向上** |
| ヒューマンエラー | 貼り付けミス・見落としのリスクあり | 機械処理で一定品質を確保 | **品質向上** |

## 6. 導入に必要なもの

| 必要事項 | 現状 | 対応 |
|---------|------|------|
| Gemini API キー | 使用中 | そのまま流用可 |
| Google Drive API | 未設定 | Google アカウントで設定（無料） |
| Google Sheets API | 未設定 | 同上 |
| Claude Code | 未導入 | 導入が必要（月額 $20〜） |
| スクリプト開発 | — | Claude Code で内製作成（外部委託不要） |

| 3週目 | 現場看板の本番運用開始 |
| 4週目 | 運用確認・チェック精度の調整 |

## 9. まとめ

月4,500回の手作業と45,000行の目視確認という反復業務を、Claude Code によるスクリプト自動化で大幅に削減できる。

追加コストは月額3,000円程度であり、削減できる人件費・工数と比較して費用対効果は高い。

まず名刺処理から試験導入し、効果を確認しながら看板処理へ展開することを推奨する。
