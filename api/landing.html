Tech Pulse API

Daily aggregated tech trends from six public sources, normalized into one clean JSON API. Built for newsletter curators, content tools, dashboards, and analytics.

Endpoints

MethodPathPurpose
GET/v1/pulse/latestFull snapshot from all sources
GET/v1/pulse/trendingTop keywords across all titles today
GET/v1/pulse/sourcesItem counts per source
GET/v1/pulse/hackernewsHacker News top stories
GET/v1/pulse/github?language=PythonGitHub trending repos (optional filter)
GET/v1/pulse/reddit/{sub}Subreddit top posts
GET/v1/pulse/qiitaQiita latest items
GET/v1/pulse/zennZenn daily articles
GET/v1/pulse/devtoDev.to top articles
GET/v1/pulse/archive/{YYYY-MM-DD}Historical snapshot

Quick example

curl -H "X-RapidAPI-Key: YOUR_KEY" \
     -H "X-RapidAPI-Host: tech-pulse-api.p.rapidapi.com" \
     https://tech-pulse-api.p.rapidapi.com/v1/pulse/trending

Use cases

  • Power a daily tech newsletter — one fetch, all sources.
  • Drive a dashboard of what developers are talking about today.
  • Train trend-detection or content recommendation models.
  • Backfill historical snapshots via the archive endpoint.
Updated daily at 00:15 UTC. All data sourced from public APIs.
